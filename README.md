# Deep Learning From Scratch

> Build these 4 from scratch and you're not a beginner anymore.

## Models Implemented

| Model | Notebook | What It Teaches |
|---|---|---|
| Autoencoder | `01_autoencoder.ipynb` | Compression, feature learning |
| Variational Autoencoder (VAE) | `02_vae.ipynb` | Probabilistic modeling, generation |
| Attention Mechanism | `03_attention_mechanism.ipynb` | Transformers, sequence modeling |
| GAN | `04_gan.ipynb` | Adversarial training, image generation |

## Quick Start

```bash
pip install -r requirements.txt
jupyter notebook
```
